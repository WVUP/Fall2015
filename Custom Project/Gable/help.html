This is the EXIT button. It allows the user to exit the site and go to the homepage of the
current site.

This is the GENERAL INFO button. This displays general information about the product,
such as vendor, product ID, and SKU.

This is the FINANCIALS button. This displays information about the product's price,
how many of the item have been sold from that vendor, and whether the item is selling well.

This is the your PREFERRED CART. Items you mark as preferred (the star icon in this section) are
added to a list of preferred products. This button tells you how many preferred products are currently in your preferred list.

This is the TAGS button. This displays information about all social networking tags
associated with this item on this vendor's site.

This is the DATABUY button. This provides a quick suggestion as to whether the product
is selling well and a buy recommendation based on that information.

This is the ADD PREFERRED button. This adds the currently displayed product to your preferred list.
The quantity of items in your list can be checked using the PREFERRED CART button.

This is the PRODUCT ATTRIBUTES button. This option lists all the data attributes currently collected
in the vendor's records. This might be the most important comparison of all (if you like data...)!
